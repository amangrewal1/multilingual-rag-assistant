# Captions, Transcripts, and Audio Description

Captions are synchronized text representations of speech and meaningful non-speech audio in video. Open captions are burned into the video frame; closed captions can be toggled off.

WCAG 1.2.2 Captions (Prerecorded) requires captions for prerecorded video with audio. 1.2.4 Captions (Live) requires captions for live audio. Auto-generated captions from YouTube or Zoom are not sufficient for 1.2.2 compliance because they miss speaker changes, punctuation, and non-speech audio.

## Transcripts

A transcript is a plain-text equivalent of an audio-only or audio-visual track, typically including speaker labels and relevant sound descriptions. Transcripts help deaf-blind users (who may read via braille display) and anyone who prefers scanning text.

## Audio description

Audio description narrates important visual information that is not conveyed through dialogue. 1.2.5 Audio Description (Prerecorded) is a Level AA requirement for prerecorded video.

## File formats

WebVTT (.vtt) is the web standard for captions and subtitles and is required by the HTML `<track>` element. SRT is widely supported by desktop players but lacks styling. TTML/IMSC is used in broadcast workflows.
