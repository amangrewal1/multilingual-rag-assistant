# Cognitive Accessibility FAQ

**Q: What is plain language?**
A: Writing that is clear, concise, and direct. It uses common words, short
sentences, and an active voice. The goal is that the reader understands on
the first read.

**Q: Why do reading-level tools disagree?**
A: Different tools use different formulae (Flesch-Kincaid, SMOG, Automated
Readability Index). Use them as rough guides, not absolutes.
